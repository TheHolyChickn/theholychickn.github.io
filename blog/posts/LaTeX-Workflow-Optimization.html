LaTeX Workflow Optimization

May 15th, 2025 • Tags: LaTeX, Workflow

Over the past year or so, I have done significant work to improve my LaTeX workflow and the organization of course files and LaTeX projects on my computer. I've decided to write a few blog posts to share my setup with others, and to document how to implement and use my setup. This first article will simply be an overview of my setup, but I will write more posts to detail how to set it up, how to use it, and how to customize it. I hope to eventually make my setup extremely well-documented, such that anyone can set it up with little or no coding knowledge. I don't expect anyone like that to actually set it up, since it runs on Linux instead of Windows or Mac, but I see no harm in trying.
